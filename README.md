# JataSets
Dataset implementations of Queue (FIFO) and Stack (LIFO) data stacks for programmatic use.

## Getting Started

### Installation
```bash
npm install
```
or
```bash
yarn install
```
### Testing
```bash
npm test
```
or
```bash
yarn test
```
### Building
```bash
npm build
```
or
```bash
yarn build
```
